## 1. 개요
회의 안건과 대화 텍스트를 분석하여 누락된 논의 사항을 파악하고, 실행 가능한 액션 아이템(Ticket)과 차기 회의 안건을 자동으로 생성하는 AI 서비스입니다.

## 2. 기술 스택
- **Frontend:** Streamlit (`ui.py`)
- **Backend:** FastAPI (`app.py`)
- **Infrastructure:** Docker (Containerization)
- **AI Engine:** LLM API (OpenAI GPT 또는 Claude 등)
- **Data Storage:** 휘발성 데이터 처리 (No DB)

## 3. 핵심 워크플로우 (Workflow)
1. **입력:** 사용자가 '회의 목표/안건'과 '회의 녹취록(텍스트)'을 입력.
2. **AI 분석:** - 입력된 안건 대비 실제 논의된 내용 비교 분석.
   - 논의되지 않은(놓친) 안건 식별.
   - 대화 흐름을 바탕으로 '다음 회의'에서 다뤄야 할 안건 제안.
3. **출력:** 요약문, 결론, 액션 아이템, 차기 안건을 UI에 표시.

### 4.1. AI 분석 및 요약

### 4.3. 외부 자료 및 태깅
