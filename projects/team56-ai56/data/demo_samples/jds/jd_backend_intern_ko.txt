[포지션] 백엔드 엔지니어 인턴

우리 팀은 채용 검토 자동화와 내부 운영 효율화를 위한 AI 기반 제품을 만들고 있습니다.
이번 포지션은 FastAPI 기반 API 개발, SQLite 또는 PostgreSQL 수준의 데이터 처리,
간단한 LLM 연동, 로그 확인과 장애 대응 보조를 함께 수행할 수 있는 백엔드 인턴을 찾습니다.

[주요 업무]
- Python 기반 백엔드 API 개발 및 유지보수
- FastAPI 또는 유사 프레임워크를 사용한 엔드포인트 구현
- SQLite/PostgreSQL 쿼리 작성 및 데이터 모델 관리
- 외부 API 연동 및 응답 파싱
- GitHub를 통한 협업, 이슈 관리, 코드 리뷰 참여
- 로그 확인, 예외 처리, 기본적인 품질 개선 작업

[필수 요건]
- Python으로 프로젝트를 구현해본 경험
- REST API 개념 이해
- SQL 기본기
- Git/GitHub 사용 경험
- 문서나 결과물로 본인이 만든 것을 설명할 수 있는 사람

[우대 사항]
- FastAPI 경험
- AI/LLM API 연동 경험
- Streamlit 또는 간단한 대시보드 구현 경험
- 개인 프로젝트 또는 공개 저장소 운영 경험

[평가 포인트]
실제 구현 경험, 문제 해결 과정, 공개 산출물의 일관성, 협업 가능성, 성장 가능성
