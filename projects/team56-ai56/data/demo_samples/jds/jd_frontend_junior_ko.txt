[포지션] 프론트엔드 주니어 개발자

우리 팀은 채용 평가 결과를 직관적으로 보여주는 웹 대시보드를 만들고 있습니다.
이번 포지션은 React 기반 화면 개발, 사용자 흐름 개선, API 연동 경험이 있는 프론트엔드 개발자를 찾습니다.

[주요 업무]
- React 기반 대시보드 화면 구현
- API 연동 및 상태 관리
- 표, 카드, 상세 패널 등 정보 밀도가 높은 UI 설계
- 반응형 레이아웃 개선
- 디자이너/백엔드와 협업하여 기능 완성도 향상

[필수 요건]
- React 또는 유사 프레임워크 프로젝트 경험
- JavaScript/TypeScript 이해
- REST API 연동 경험
- Git/GitHub 협업 경험

[우대 사항]
- 데이터 시각화 경험
- 접근성, UX 개선 경험
- 디자인 시스템 또는 컴포넌트 단위 개발 경험

[평가 포인트]
UI 구현력, 사용자 관점, 협업 방식, 공개 산출물의 품질, 설명력
