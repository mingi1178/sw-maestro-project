# 팩폭머니 (FactPokMoney) — UI 개발 BRIEF

> AI Agent 금융 코칭 챗봇 — 프론트엔드 UI

---

## 1. 컨셉

**팩폭머니** = Fact + 팩폭(팩트로 폭행) + Money

사용자의 소비 데이터를 기반으로 **직설적이고 단호한 톤**으로 금융 코칭을 제공하는 챗봇.
두루뭉술한 위로가 아닌, **숫자와 사실에 근거한 팩트 지적**으로 소비 습관 개선을 유도한다.

- 기존 금융 앱: "이번 달 지출이 조금 많네요 😊"
- 팩폭머니: "이번달에 소확행에 21번 총 80만원을 썼어요. 이건 '소소'하다고 볼 수 없어요."

---

## 2. 담당 범위

**포함**: 채팅 UI 화면 구성, 컴포넌트 스타일링, 더미데이터 연동
**제외**: 백엔드 API 호출 로직(다른 프론트 담당), AI Agent 연동(백엔드 담당), 그래프 렌더링(별도 담당 — **자리만 확보**)

---

## 3. 기술 스택

- **프레임워크**: Next.js (App Router) + TypeScript
- **스타일링**: Tailwind CSS
- **UI 컴포넌트**: shadcn/ui
- **마크다운**: react-markdown + remark-gfm
- **상태 관리**: Zustand


---

## 4. 화면 구성

**단일 챗봇 화면 1개** (라우팅 불필요).

구성 요소:
- **헤더**: 프로젝트명 표시
- **메시지 영역**: AI 메시지(좌)·사용자 메시지(우), 자동 스크롤
- **그래프 자리**: AI 메시지 직하단에 빈 컨테이너 (별도 담당자가 채울 공간)
- **입력 영역**: 텍스트 입력창 + 전송 버튼 + **파일 첨부 버튼(CSV용)**

### AI 응답 출력 방식
백엔드는 완성된 텍스트를 한 번에 응답하지만, **UI에서 한 글자씩 타이핑되듯 출력**한다 (프론트 단 흉내내기). 응답 대기 중에는 타이핑 인디케이터(점 3개) 표시.

---

## 5. 디자인

**컨셉**: 보라색 + 그라데이션 / 세련됨, 모던, 신뢰감

- **배경**: 연보라 그라데이션
- **강조 요소** (금액·기간·횟수 등 숫자, 포인트 UI): 진보라 그라데이션

## 6. 더미데이터

### 6.1 CSV 파일

- **경로**: `Team41_Vector/sample-data/persona_A_transactions.csv` (1개월치 소비내역)
- 코드 작성 전 **파일을 직접 열어 컬럼 구조·데이터 타입·인코딩을 확인**한 뒤 타입 정의 및 파싱 로직 작성. 추측 금지.

### 6.2 챗봇 응답 더미 메시지

#### 톤앤매너
- 존댓말 / 직설적 / 돌려 말하지 않음
- **금융적 사실·숫자만** 근거로 사용
- 인종·성별·외모 비하, 자학 유도, 도덕적 비난 ❌
- 핵심 기법: 구체적 수치 제시 / 모순 짚기 / 비교 데이터 활용

#### 샘플 메시지

1. 이번 달 배달앱 결제 17건, 총 34만 2천원이에요. 같은 기간 마트 장보기는 2건이고요. '집밥 해먹겠다'고 하신 게 3주 전입니다.
2. 카페 지출이 월급의 8%예요. 평균적인 가계의 식비 전체 비중이 12~15%인 점을 고려하면, 카페만으로 식비의 절반을 쓰고 계신 셈이에요.
3. 최근 3개월간 한 번도 사용하지 않은 구독 서비스가 4개 있어요. 월 합계 5만 8천원, 연간 69만 6천원입니다.
4. 동일 연령대·소득대 평균 외식비는 월 28만원이에요. 회원님은 이번 달 62만원을 쓰셨습니다. 평균의 2.2배입니다.
5.  '어쩌다 보니'라고 하셨는데, 동일 패턴이 지난 4개월 연속 반복되고 있어요. '어쩌다'가 아니라 습관입니다.
