# 팩폭머니 데모 시나리오

샘플 데이터: `sample-data/transactions_2026_mar_jun.csv` (2026년 3~6월, 319건)

---

## 시나리오 흐름

```
① CSV 업로드 → ② 자동 분석 팩폭 → ③ 사용자 변명 → ④ 코치 반박
→ ⑤ 그래프 요청 → ⑥ 비교 분석 → ⑦ 미션 수락 → ⑧ 미션 완료
```

---

## ① CSV 업로드

채팅 입력창 왼쪽 클립 아이콘 클릭 → `transactions_2026_mar_jun.csv` 첨부 → 전송

> 파일 업로드 즉시 자동 분석이 시작됩니다.

---

## ② 자동 분석 — 예상 코치 응답

> 기대 포인트: 쇼핑·배달 수치 직격, 월별 증가 추이 지적

```
4개월 데이터 분석했습니다. 총 지출이 3월 181만원에서 6월 212만원으로
**4개월 만에 17% 증가**했습니다.

쇼핑에만 3월 359,000원 → 5월 479,000원으로 **33% 급증**했고,
무신사와 29CM에서만 **4개월 합산 1,109,000원** 나갔습니다.
배달은 매달 312,000원대로 고정 — 사실상 고정 지출입니다.
```

카테고리 차트 + 자산 시뮬레이션 차트가 메시지 아래 자동 표시됩니다.

## ④ 코치 반박 — 예상 응답

> 기대 포인트: 말꼬리 잡기 + 비꼬기

```
줄이고 계신다고 하셨는데, 5월 지출이 4월 대비 **8% 증가**했습니다.
카페는 스타벅스 강남R점 혼자서 **24회, 139,200원**이고요.
이 정도면 스타벅스 주주 하셔도 되겠습니다.
```

## ⑤ 일별 소비 그래프 요청

```
5월 소비 그래프 보여줘
```

> `get_daily_spending("2026-05")` 툴 호출 → 꺾은선 그래프 렌더링  
> 카테고리별 개별 선 + 총합 굵은 선이 함께 표시됩니다.

---

## ⑥ 3개월 비교 요청

```
저번달이랑 그 전달이랑 비교해줘
```

> `get_spending_comparison()` 툴 호출  
> 이번달·저번달·저저번달 카테고리별 + 가맹점별 지출을 비교 분석합니다.

```
팝업스토어 지출이 3월 180,000원 → 6월 285,000원으로 **3개월 연속 증가** 중입니다.
배달은 매달 312,600원으로 거의 동일 — 이미 고정 지출로 굳었습니다.
```

## 핵심 어필 포인트

| 포인트 | 내용 |
|--------|------|
| **팩폭 톤** | 수치 직격 + 말꼬리 잡기 + 비꼬기 |
| **Tool Calling** | LLM이 필요 시 스스로 DB 조회 (5종 도구) |
| **인터랙티브 미션** | 수락 → 완료 → 폭죽까지 한 흐름 |
| **그래프 온디맨드** | 요청할 때만 꺾은선 그래프 렌더링 |
| **타이핑 애니메이션** | 백엔드 일괄 응답을 프론트에서 글자별 재현 |
