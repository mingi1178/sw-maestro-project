# twin-match

AI persona-based dating simulation platform built with a FastAPI backend and a Next.js frontend.

## Structure

- `backend/`: FastAPI backend for agent, conversation, chemistry, and job workflows
- `frontend/`: Next.js frontend for the user-facing dating simulation experience

## Backend

- Stack: FastAPI, SQLAlchemy async, SQLite, Upstage Solar LLM
- Setup details: `backend/README.md`

## Frontend

- Stack: Next.js, React, TypeScript, Tailwind CSS, TanStack Query
- Setup details: `frontend/README.md`
