# soma-recommender-extension

SW Maestro 연수생을 위한 개인 맞춤 특강 추천 Chrome 확장 프로그램입니다.

## Development

의존성을 설치합니다.

```bash
npm install
```

코드 품질을 확인합니다.

```bash
npm run lint
npm run format:check
```

코드 포맷을 적용합니다.

```bash
npm run format
```

## Chrome Extension Load

1. Chrome에서 `chrome://extensions`로 이동합니다.
2. 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 선택합니다.
4. 이 프로젝트 루트 폴더를 선택합니다.
