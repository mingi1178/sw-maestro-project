03LangGraph 에이전트 노드 흐름

시스템 프롬프트로 ReAct 3단계(일정→건강→운동)를 강제한다. 멀티턴 수정 요청은 체크포인터에서 이전 상태를 복원해 refine_node로 진입한다. 진입점은 run_agent_stream(user_input, thread_id) — async iterator로 ChatChunk를 yield한다.
